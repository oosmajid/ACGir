# اجرای ساده روی ویندوز

برای کاربر نهایی، خروجی آماده داخل پوشه `dist\ACcrawler` قرار می‌گیرد. کافی است کل همین پوشه کپی یا zip شود و فایل `ACcrawler.exe` اجرا شود.

این خروجی به نصب Python، ساختن `venv`، نصب پکیج‌ها یا نصب جداگانه `ffmpeg` نیاز ندارد.

## ساخت نسخه ویندوز

ساخت فایل exe باید روی ویندوز انجام شود، چون PyInstaller خروجی ویندوز را از macOS نمی‌سازد.

1. روی یک سیستم ویندوزی، فقط برای مرحله ساخت، Python 3.11 یا جدیدتر نصب باشد.
2. فایل `build_windows.bat` را اجرا کنید.
3. بعد از پایان ساخت، فایل `dist\ACcrawler-portable.zip` یا پوشه `dist\ACcrawler` را به کاربر نهایی بدهید.

برنامه از رابط سبک `tkinter` استفاده می‌کند و دیگر Gradio یا مرورگر داخلی اجرا نمی‌شود، بنابراین build باید خیلی سبک‌تر باشد.

نکته: فایل `ACcrawler.exe` را از پوشه خودش جدا نکنید. کل پوشه `dist\ACcrawler` باید کنار هم بماند، چون فایل‌های داخلی برنامه و نسخه همراه ffmpeg داخل همان پوشه قرار دارند.
